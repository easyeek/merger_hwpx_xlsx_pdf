비즈니스 데이터(한글, 엑셀, PDF) 통합 및 행정 자동화 솔루션: Premium Merger v2.0

1. 프로젝트 개요 및 배경

파편화된 한글 문서, 방대한 엑셀 데이터, 그리고 고정된 형태의 PDF 파일을 하나의 마스터 파일로 자동 통합하여 비즈니스 전처리 시간을 획기적으로 개선한 종합 자동화 프로젝트입니다.

  • 📅 최종 업데이트 : 2026.03
  • 👤 주요 역할 : 통합 대시보드 설계, 이중 인코딩 폴백 엔진 및 고속 PDF 병합 로직 구현, 프리미엄 UI 디자인 시스템 확립
  • 🚩 도입 배경 : 포맷별로 분산된 툴의 통합 필요성 증대 및 대용량 데이터 처리 시의 안정성 확보
  • 🎯 해결 목표 : 프리미엄 UX 기반의 단일 통합 인터페이스를 통한 업무 누수 제로화 및 배포 편의성 극대화

2. 기술 스택 및 환경

분류 적용 기술 및 도구 도입 목적
언어/환경 Python 3.12, venv 최신 문법 활용 및 가상환경을 통한 의존성 독립성 확보
데이터 분석 Pandas, NumPy, openpyxl 대용량 엑셀 데이터의 논리적 통합 및 전처리 연산 가속화
오피스/PDF pywin32, pypdf 한글 OLE Automation 및 고속 PDF 스트림 병합 라이브러리 활용
GUI/Build Tkinter, PyInstaller 프리미엄 다크 모드 인터페이스 구축 및 단일 EXE 배포 환경 구성

4. 주요 해결 전략

전략 1: [이중 인코딩 폴백 및 데이터 무결성]

엑셀/CSV 통합 시 UTF-8과 CP949를 자동으로 교차 검증하는 인코딩 폴백 시스템을 구축하여 데이터 깨짐 문제를 해결하고 무결성을 100% 확보했습니다.

전략 2: [멀티스레딩 기반 비정기적 UI 제어]

대용량 문서 처리 시 GUI가 멈추는 프리징 현상을 방지하기 위해 연산 로직을 백그라운드 스레드로 분리하고, 실시간 진행 상태를 Progressbar로 시각화했습니다.
