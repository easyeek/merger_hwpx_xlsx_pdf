# Premium Merger v2.0 🚀

> **한글(HWPX), 엑셀(XLSX/CSV), PDF 문서를 스마트하게 통합하는 프리미엄 데이터 병합 도구**

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.12-yellow.svg)
![easyeek](https://img.shields.io/badge/created_by-easyeek-gold.svg)

## 🌟 주요 기능

- **📄 한글 문서 병합**: 여러 개의 HWP/HWPX 파일을 하나의 문서로 수직 통합 (서식 유지).
- **📊 엑셀 데이터 통합**: 파일별 시트 결합 또는 모든 시트 자동 결합 지원, 인코딩 자동 감지.
- **📕 PDF 문서 합치기**: 고속 PDF 병합 라이브러리를 통한 안정적인 병합.
- **💎 프리미엄 UI**: 다크 모드 기반의 세련된 GUI 대시보드 제공.
- **📦 단일 EXE 배포**: 별도의 파이썬 설치 없이 즉시 실행 가능.

## 🛠 기술 스택

- **Core**: Python 3.12, venv
- **Data**: Pandas, Openpyxl, pypdf
- **Automation**: pywin32 (HWP OLE Automation)
- **GUI**: Tkinter (Premium Dark Theme)
- **Build**: PyInstaller

## 🚀 시작하기

### 1. 실행 파일 사용 (권장)
`dist/Premium_Merger_v2.exe` 파일을 다운로드하여 직접 실행하세요.

### 2. 소스 코드 실행
```bash
# 가상환경 생성 및 활성화
python -m venv .venv
source .venv/Scripts/activate  # Windows

# 의존성 설치
pip install -r requirements.txt

# 실행
python main.py
```
